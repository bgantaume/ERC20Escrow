# ERC20 Escrow

Escrow is a contract designed to garantee a safe transfer beetween a sender and a recipient.

Typical usage :
1. Buyer locks in some token
2. Merchant send the goods
3. Buyer releases the tokens
4. Merchant claims the tokens
